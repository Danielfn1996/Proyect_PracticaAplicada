descargar git

click derecho git bash here 

y ingresa los siguientes comandos


$ git config --global user.name "cristian rojas"				// SU NOMBRE

$ git config --global user.email "crrojasruiz@poligran.edu.co"			// CORREO CON EL QUE ESTA INSCRITO EN GITHUB DEL REPOSITORIO
